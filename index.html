General

Inspired by Stephen Few and his book "Information Dashboard Design: Displaying Data for At-a-Glance Monitoring" we have created this sample which is based on FIGURE 13.15 A sample CIO dashboard.

Replacing Data

To use real data replace data in /src/js/data.js.
